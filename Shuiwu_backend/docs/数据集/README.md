# 税小通微调数据集说明

## 数据集概述

本数据集专为税小通税务智能助手的本地大模型微调而设计，包含800条高质量的Alpaca JSONL格式训练数据。

## 数据集结构

```
数据集/
├── identity_data.jsonl          # 身份数据集（400条）
├── general_data.jsonl           # 通用数据集（400条）
└── shuixiaotong_finetune_800.jsonl  # 完整数据集（800条）
```

## 数据格式

采用Alpaca JSONL格式，每条数据包含三个字段：

```json
{
  "instruction": "问题或指令",
  "input": "输入内容（本数据集中为空）",
  "output": "回答或输出"
}
```

## 数据分布

### 1. 身份数据集（400条）

#### 1.1 自我介绍类（150条）
- 身份定义、功能介绍、能力说明
- 涵盖：我是谁、我能做什么、我的专业领域等

#### 1.2 公司信息类（150条）
- 税小通平台介绍、服务范围、会员体系
- 涵盖：服务内容、收费标准、平台功能等

#### 1.3 模型定位类（100条）
- 专业定位、服务边界、核心价值
- 涵盖：与通用AI的区别、服务局限性、专业性保证等

### 2. 通用数据集（400条）

#### 2.1 行业任务类（150条）
- **增值税相关**：税率、计算、抵扣、优惠政策等
- **企业所得税相关**：税率、扣除、优惠政策等
- **个人所得税相关**：税率、专项附加扣除、汇算清缴等
- **税务筹划相关**：筹划原则、优惠政策、风险防控等

#### 2.2 文本处理类（150条）
- **合同审查**：税务风险识别、发票条款审查、风险点分析等
- **风险识别**：常见风险点、风险防范、稽查应对等
- **报告生成**：企业体检报告、税负分析、合规检查等
- **发票管理**：开具要求、丢失处理、异常凭证处理等

#### 2.3 知识问答类（100条）
- **政策解读**：最新税收政策、优惠政策、政策变化等
- **实务操作**：办税流程、申报操作、申请程序等
- **税务争议**：行政复议、稽查应对、处罚处理等

## 数据特点

### 1. 专业性强
- 基于税小通项目的实际Agent提示词生成
- 涵盖增值税、企业所得税、个人所得税等主要税种
- 包含税务筹划、风险防控、合同审查等专业领域

### 2. 覆盖全面
- 身份数据：明确模型定位和服务边界
- 行业数据：覆盖主要税种和业务场景
- 实务数据：包含操作指导和风险防控

### 3. 格式规范
- 严格遵循Alpaca JSONL格式
- 问答对清晰，适合指令微调
- 中文内容，符合中国税务实务

## 使用建议

### 1. 微调参数建议
- 学习率：1e-5 ~ 5e-5
- 批次大小：根据GPU显存调整
- 训练轮数：3-5轮
- 优化器：AdamW

### 2. 数据增强建议
- 可结合税务法规文档进行增量预训练
- 可添加更多行业特定场景数据
- 可结合实际用户对话数据进行优化

### 3. 评估建议
- 使用税务知识问答测试集评估
- 测试模型对身份认知的准确性
- 评估专业税务问题的回答质量

## 数据来源

本数据集基于以下项目资源生成：
- 税小通Agent提示词系统
- 税务知识库内容
- 税收政策法规文档
- 实务操作指南

## 注意事项

1. 本数据集仅用于模型微调训练
2. 数据内容基于中国税收政策，适用于中国税务场景
3. 政策时效性：部分政策可能有更新，使用时需注意政策变化
4. 专业性：模型回答仅供参考，具体税务处理请咨询专业机构
